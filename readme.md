# Make-txt-to-SPCD

**中文** | [English](README_en.md)

## 简介
本项目是一个专门用于将包含光谱数据的txt文件转换为OpticStudio SPCD格式的工具。它提供了一个简单易用的图形界面，让用户可以方便地将光谱数据转换为光学设计软件所需的格式。

![snapshot](./data/snapshot.png)

## 特性
- 支持将光谱数据txt文件转换为OpticStudio SPCD格式
- 提供直观的图形用户界面
- 支持多种波长单位（纳米nm、微米μm）
- 可选择对强度进行归一化处理
- 支持限制数据行数（可选10、20、50、100、200行）
- 自动在标题注释中包含原文件名和转换日期
- 实时绘制光谱曲线预览
- 支持自定义输出文件路径

## 依赖
在使用本工具前，请确保已安装以下Python包：

```bash
pip install FreeSimpleGUI
pip install matplotlib
```

## 使用方法

直接运行脚本即可启动图形界面：
```bash
python txt2spcd_GUI.py
```

在图形界面中：
1. 点击"浏览"按钮选择输入的txt文件
2. 点击"保存"按钮指定输出的spcd文件路径
3. 从下拉菜单中选择数据行数（10、20、50、100、200）
4. 选择波长单位（nm或μm）
5. 选择是否归一化强度
6. 点击"转换"按钮开始转换

## 输入文件格式
输入的txt文件应包含两列数据：
- 第一列：波长值（数值）
- 第二列：强度值（数值）
- 可以使用空格或制表符分隔两列
- 支持以#开头的注释行

示例：
```
400 0.1
450 0.3
500 0.7
550 0.9
600 0.6
650 0.2
700 0.05
```

## 输出文件格式
输出的SPCD文件遵循OpticStudio标准格式：
- 波长以微米（μm）为单位
- 最多200行数据
- 两列：波长（浮点数）、相对强度（浮点数）
- 支持0-1范围内的强度归一化

## 注意事项
- 输入文件必须包含有效的数值数据
- 波长值会被自动转换为微米单位（OpticStudio要求）
- 如需处理超过200行的数据，可通过下拉菜单选择合适的行数进行等间距采样
- 强度归一化将把最大值设为1，其他值按比例缩放
